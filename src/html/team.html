Meet the Programmers

In the days leading to the end of July 2022, the Instructor volunteered to take students interested in the world of tech some little bit of code.

It started with Python but for the need to enable students make projects they could easily showcase , HTML , CSS and a little bit of JavaScript for functionality was arrived at preparing them for a career in web development.

In the days leading up to the completion-including the summer holidays- of this project in October 2022, the students learnt firsthand from bottom up. First by working on smaller projects like Calculator design, Chess Board with css grids , redesign of landing pages of selected websites which ultimately birthed developing one for the use of the school.

The website was made with the use of the following resources with some used for selected parts of the developemt process : HTML5, CSS3, JS, JQuery with Git/Github used for Version Control, Netlify for site hosting and NetlifyCMs and Gatsby JS for the Blog page.

While there is a lot of improvement to be made to this website, it is in the best hope that we these students will be the ones to make those improvements as they grow in their knowledge of languages , frameworks and libraries.

We believe that with the early exposure to the world of tech, the students can go on to pursue careers as FrontEnd Engineers, Backend Engineers , FullStack Engineers, Data Scientists , Machine Learning Engineers or any of the other vast career options the IT world provides today.

With this project we hope to have helped the students realise they are not hindered by traditional career options and there is a lot more they could possibly become .
